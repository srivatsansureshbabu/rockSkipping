# 🪨 Rock Skipping Game

A simple 3D rock skipping game built with Ursina Engine and Python.
You control a player on the beach, pick up a rock, and throw it across the ocean to see how far it flies!

# 🎮 Features

✅ First-person movement (WASD + mouse look)
✅ Throw rocks by holding and releasing space bar
✅ Rocks fly forward and return to your hand
✅ Ocean background and beach environment
